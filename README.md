## Project Goal

The goal of this project is to develop a machine learning-based system capable of detecting spam and phishing messages using Natural Language Processing (NLP) techniques. The system will classify messages as either legitimate (ham) or malicious (spam/phishing) and evaluate multiple machine learning models to determine the most effective approach.

## Dataset

Email Spam Detection Dataset (Kaggle)

Dataset Link:
https://www.kaggle.com/datasets/abdmental01/email-spam-dedection
